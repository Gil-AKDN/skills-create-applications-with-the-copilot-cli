## CLI calculator

This repository now includes a basic Node.js CLI calculator in `src/calculator.js`.

Supported operations:
- addition: `+`
- subtraction: `-`
- multiplication: `*`, `x`, `X`, `×`
- division: `/`, `÷`

Run it with:

```bash
npm start -- 8 + 4
```

Or:

```bash
node src/calculator.js 8 + 4
```
